README

Ce répertoire contient les codes pour les simulations utilisant les deux  conditions  de Dirichlet et Robin.

1. Condition de Dirichlet:
   - Emplacement : code_dirichlet/
   - Pour compiler le code, naviguez dans le dossier 'code_dirichlet' et exécutez la commande 'make'.
   - Pour exécuter la simulation, utilisez la commande suivante :
     mpirun -np 4 simulation
   Vous pouvez remplacer '4' par le nombre de processus que vous souhaitez utiliser.

2. Condition de Robin:
   - Emplacement : code_robin/
   - Pour compiler le code, naviguez dans le dossier 'code_robin' et exécutez la commande 'make'.
   - Pour exécuter la simulation, utilisez la commande suivante :
     mpirun -np 4 simulation
   Vous pouvez remplacer '4' par le nombre de processus que vous souhaitez utiliser.
